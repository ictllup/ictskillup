What is Computer Hardware?

Computer hardware refers to the physical components of a computer system.

Types of Computer Hardware

  • Input Devices: Keyboard, mouse, scanner, webcam, microphone, etc.
  • Processing Devices: Central Processing Unit (CPU), Graphics Processing Unit (GPU), etc.
  • Storage Devices: Hard Disk Drive (HDD), Solid State Drive (SSD), Flash Drive, etc.

  • Peripheral Devices: Modem, router, network card, etc.

Components of a Computer System

  • CPU (Central Processing Unit): The brain of the computer, responsible for executing instructions.
  • Motherboard: The main circuit board of the computer, connects all hardware components together.
  • Memory (RAM): Temporary storage for data and programs, volatile memory.
  • Storage Devices: Permanent storage for data and programs, non-volatile memory.
  • Power Supply: Converts AC power from the mains to DC power for the computer components.

Functions of Computer Hardware

  • Input: Receiving data and instructions from the user.
  • Processing: Executing instructions and performing calculations.
  • Storage: Storing data and programs.
  • Output: Displaying or printing the results of processing.
  • Control: Managing the flow of data and instructions.

Meaning of Computer Hardware

Computer hardware refers to the physical components of a computer system.

Classifications of Hardware

  • Input Hardware: Devices that allow users to enter data or instructions into the computer.

Keyboard

A keyboard is an input device that allows users to enter text, numbers, and commands into the computer.

Functions of Some Keyboard Buttons

  • Enter: Inserts a new line or executes a command.
  • Backspace: Deletes the character to the left of the cursor.
  • Spacebar: Inserts a space between words.
  • Shift: Changes the case of letters or activates alternate keys.
  • Ctrl (Control): Used in combination with other keys to execute commands.
  • Alt (Alternate): Used in combination with other keys to execute commands.
  • Delete: Deletes the character at the cursor position.

Common Keyboard Shortcuts

  • Ctrl + C: Copy
  • Ctrl + X: Cut
  • Ctrl + V: Paste
  • Ctrl + Z: Undo
  • Ctrl + A: Select all
  • Ctrl + S: Save
  • Ctrl + P: Print
  • Alt + Tab: Switch between open windows
  • F1: Help
  • F5: Refresh

Computer Assembly Guide

Preparation

  • Ground yourself: Static electricity can damage components. Touch a grounded metal object or wear an anti-static wrist strap.
  • Unpack and prepare components:
    • CPU (Central Processing Unit)
    • Motherboard
    • RAM (Memory)
    • Storage Drive (HDD or SSD)
    • Power Supply
    • Graphics Card (if separate)
    • Case
    • Cooling System (CPU Cooler and Case Fans)
    • Operating System
  • Read manuals and instructions: Familiarize yourself with component manuals and instructions.

Step 1: Unpack and Prepare the Case

  1. Remove side panels: Take out screws and gently pull out side panels.
  2. Install case fans: Mount fans to case using screws.
  3. Install front panel headers: Connect front panel headers (power, reset, LEDs) to motherboard.

Step 2: Install Motherboard

  1. Unpack motherboard: Remove from anti-static bag and prepare for installation.
  2. Install motherboard stand-offs: If required, install stand-offs to support motherboard.
  3. Mount motherboard: Align motherboard with case and secure with screws.
  4. Connect motherboard cables:
    • Power cables (ATX, EPS)
    • Data cables (SATA, USB)
    • Fan cables

Step 3: Install CPU and CPU Cooler

  1. Unpack CPU: Handle by edges to prevent damage.
  2. Apply thermal paste: If required, apply a thin layer to CPU die (center).
  3. Install CPU: Gently insert CPU into socket, ensuring alignment.
  4. Secure CPU: Close socket lever or apply even pressure.
  5. Install CPU Cooler: Follow manufacturer's instructions.

Step 4: Install RAM

  1. Identify RAM slots: Locate slots on motherboard.
  2. Install RAM: Align notches and gently push RAM into slots.

Step 5: Install Storage Drive

  1. Unpack storage drive: Remove from anti-static bag.
  2. Install storage drive: Mount to case using screws.
  3. Connect storage drive cables:
    • SATA
    • Power

Step 6: Install Power Supply

  1. Unpack power supply: Remove from anti-static bag.
  2. Install power supply: Mount to case using screws.
  3. Connect power supply cables:
    • ATX
    • EPS
    • Peripheral cables

Step 7: Install Graphics Card (if separate)

  1. Unpack graphics card: Remove from anti-static bag.
  2. Install graphics card: Align with PCIe slot and gently push in.
  3. Secure graphics card: Use screws to secure to case.

Step 8: Connect Front Panel and Fans

  1. Connect front panel headers: Connect to motherboard.
  2. Connect fan cables: Connect to motherboard or power supply.
