  • 21-30 Short Answer
  • 21. ICT stands for Information Communication Technology, which refers to the use of technology to manage, process, and store information.
  • 22. Examples of ICT tools include computers, smartphones, internet, social media, and software applications.
  • 23. The internet is a global network of computers connected together to share information.
  • 24. A computer network is a group of computers connected together to share resources and information.
  • 25. A database is a collection of organized data that can be easily accessed and managed.
  • 26. Software is a set of instructions that tells a computer what to do.
  • 27. Hardware refers to the physical components of a computer system.
  • 28. A motherboard is the main circuit board of a computer that connects all hardware components together.
  • 29. A CPU (Central Processing Unit) is the brain of a computer that executes instructions.
  • 30. A keyboard is an input device that allows users to enter data into a computer.
