Introduction to Word Processing

Purpose of Word Processing

  • Word processors are mostly used for writing letters, reports, projects, books, essays, memos, resumes (CVs), etc.

Word Processing Overview

Definition

Word processing is the art of creating, saving, editing, formatting, and printing text and graphic documents using electronic word processors.

Word Processors

Word processors (word processing software) are programs used for creating, editing, saving, and printing documents. They are used to create documents such as letters, labels, reports, Curriculum Vitae (CVs), and proposals.

Examples of Word Processors

  • Microsoft Word
  • Corel WordPerfect
  • Lotus WordPro
  • Apple Pages
  • OpenOffice.org Writer
  • WordStar
  • Apple iWork
  • Zoho Writer
  • Google Docs
  • LibreOffice Writer
  • AbiWord
  • WPS Office

Text Editors

Text editors are simple word processors that are generally used to type without any special formatting. They are mainly used to create small notes, memos, and programs.

Examples of Common Text Editors

  • Notepad
  • Notepad++
  • Gedit
  • WordPad
  • JCreator

Basic Terminology

  • Typeface: The shape of the characters, e.g., Times New Roman, Arial, Tahoma.
  • Line Spacing: The amount of vertical white space between two lines of text, measured in points.
  • Text Alignment: How lines of text are arranged relative to the margins - left, center, right, or justify.
  • Indent: The white space set between the margin and the beginning of text.
  • Formatting Text: Changing the appearance of text using commands like bold, italics, underlining, font color, etc.
  • Editing Text: Making changes to the content of an existing document using cut, copy, paste, overtype, undo, insert, delete.
  • Clipboard: An area of memory to temporarily store copied or cut text, graphics, etc., before pasting.
  • Header/Footer: Text that appears in the top/bottom margin of all pages.
  • Ruler: Used to set indent, margin, and tab markers.
  • Hard Copy: A printed document on physical paper.
  • Soft Copy: A document stored on a disk or other computer storage device.
  • Overtype Mode: Replaces characters at the cursor as you type. Toggle with Insert key.
  • Paragraph: Text between one paragraph break (Enter key) and the next.
  • Proofreading: Reviewing a document for accuracy using spelling/grammar check, thesaurus, etc.
