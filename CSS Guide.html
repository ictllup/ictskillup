Introduction to CSS

CSS (Cascading Style Sheets) is a language used to describe the presentation of web pages, including layout, colors, and fonts. It helps separate content (HTML) from design, making websites easier to maintain and style.

CSS enhances the user experience by allowing for consistent styling across multiple web pages and by making it easier to apply and maintain styles.

Types of CSS

Inline CSS

Inline CSS involves applying styles directly within an HTML element using the style attribute.

<h1 style="color: blue;">This is a heading</h1>

Best for quick, specific styling on a single element.

Internal (Embedded) CSS

Internal CSS is written inside a <style> tag within the <head> section of an HTML document.

<head>
    <style>
        h1 {
            color: blue;
        }
    </style>
</head>

Ideal for styling a single page when external stylesheets are not necessary.

External CSS

External CSS is written in a separate file with a .css extension and linked to the HTML document using the <link> tag.

<head>
    <link rel="stylesheet" href="styles.css">
</head>

Best for applying consistent styles across multiple pages, enabling easier maintenance and organization of styles.

CSS Syntax

The basic structure of CSS consists of selectors, properties, and values.

p {
    color: blue; /* Property: color, Value: blue */
    font-size: 16px; /* Property: font-size, Value: 16px */
}

Selectors

Element Selector

Targets all instances of a specific HTML element.

p {
    color: blue;
}

Class Selector

Targets elements with a specific class attribute.

.myClass {
    color: green;
}

ID Selector

Targets a single element with a specific ID attribute.

#myID {
    color: red;
}

Combinators

Define relationships between elements.

Descendant Selector

div p {
    color: gray; /* Selects all <p> inside <div> */
}

Child Selector

div > p {
    color: black; /* Selects direct children <p> of <div> */
}

Pseudo-Classes

Apply styles based on the state of an element.

a:hover {
    color: orange; /* Changes link color when hovered */
}

Pseudo-Elements

Style specific parts of an element.

p::first-line {
    font-weight: bold; /* Styles the first line of a paragraph */
}

The CSS Box Model

The box model is a fundamental concept that dictates the space an element occupies on a web page. It consists of:

  • Content: The actual content inside the element (e.g., text, images).
  • Padding: Space between the content and the element's border.
  • Border: A line surrounding the padding (or content if no padding is set).
  • Margin: Space outside the border, separating the element from neighboring elements.
div {
    width: 200px;
    padding: 10px;
    border: 5px solid black;
    margin: 20px;
}

Background Design

Background Color

The background-color property sets the background color of an element.

body {
    background-color: #f4f4f4; /* Light gray background */
}

Background Image

Use the background-image property to set an image as the background.

body {
    background-image: url('images/background.jpg');
    background-size: cover; /* Ensures the image covers the entire background */
    background-repeat: no-repeat; /* Prevents the image from repeating */
    background-position: center center; /* Centers the image */
}

Gradient Backgrounds

Gradients can create smooth transitions between colors using linear-gradient() or radial-gradient().

body {
    background-image: linear-gradient(to right, #ff7e5f, #feb47b); /* Gradient from left to right */
}

Typography and Font Design

Font Family

The font-family property specifies the typeface to be used.

body {
    font-family: 'Arial', sans-serif;
}

h1 {
    font-family: 'Georgia', serif;
}

Font Size

The font-size property sets the size of the font.

p {
    font-size: 16px; /* Standard size for paragraphs */
}

h1 {
    font-size: 36px; /* Larger font size for headings */
}

Font Weight

The font-weight property controls the thickness of the text.

h1 {
    font-weight: bold; /* Bold font */
}

p {
    font-weight: normal; /* Normal font weight */
}

Font Style

The font-style property can be used to italicize text.

em {
    font-style: italic; /* Italic text */
}

Line Height

The line-height property controls the space between lines of text, enhancing readability.

p {
    line-height: 1.6; /* Adds space between lines for better readability */
}

Text Align

The text-align property specifies the horizontal alignment of text within an element.

h1 {
    text-align: center; /* Centers the heading */
}

p {
    text-align: justify; /* Justifies the paragraph text */
}

Layout with Flexbox

Flexbox (Flexible Box Layout) is a CSS layout model designed to distribute space along a single axis (either horizontal or vertical) and align items within a container.

Key Properties

  • display: flex;: Defines a flex container and enables flex context for all its children.
  • justify-content: Aligns items along the main axis (default is horizontal).
  • flex-start: Aligns items to the start of the container.
  • center: Centers items.
  • space-between: Distributes items with space between them.
  • space-around: Distributes items with space around them.
  • align-items: Aligns items along the cross axis (default is vertical).
  • flex-start: Aligns items to the start of the container.
  • center: Centers items.
  • flex-end: Aligns items to the end of the container.
  • stretch: Stretches items to fill the container.
<div class="container">
    <div class="item">Item 1</div>
    <div class="item">Item 2</div>
    <div class="item">Item 3</div>
</div>
.container {
    display: flex;
    justify-content: space-between; /* Align items horizontally with space between */
    align-items: center; /* Align items vertically in the center */
    background-color: #f4f4f4;
}

.item {
    flex: 1; /* Each item will grow equally */
    background-color: #fff;
    padding: 10px;
    border: 1px solid #ddd;
}

Grid Layout

CSS Grid Layout provides a two-dimensional grid-based layout system that lets you design complex layouts easily.

Key Properties

  • display: grid;: Defines a grid container.
  • grid-template-columns: Defines the number and size of columns.
  • repeat(3, 1fr): Creates 3 columns, each taking up one fraction of the available space.
  • grid-template-rows: Defines the number and size of rows.
  • gap: Specifies the space between grid items.
<div class="grid-container">
    <div class="grid-item">1</div>
    <div class="grid-item">2</div>
    <div class="grid-item">3</div>
</div>
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* 3 equal columns */
    gap: 10px; /* Space between grid items */
}

.grid-item {
    background-color: #f4f4f4;
    padding: 20px;
    border: 1px solid #ddd;
}

Styling

Colors and Backgrounds

body {
    background-color: #f4f4f4; /* Light gray background for the entire page */
}

.header {
    background-color: #333; /* Dark background for header */
    color: white; /* White text color */
    padding: 10px 20px; /* Space inside header */
}

Fonts

h1 {
    font-family: 'Arial', sans-serif; /* Font family for headings */
    font-size: 2em; /* Larger font size for headings */
    color: #333; /* Dark gray text color */
}

p {
    font-family: 'Georgia', serif; /* Font family for paragraphs */
    font-size: 1em; /* Standard font size for paragraphs */
    line-height: 1.6; /* Increased line height for readability */
}

Spacing

.container {
    margin: 0 auto; /* Center container horizontally */
    padding: 20px; /* Space inside container */
}

.box {
    margin-bottom: 15px; /* Space below each box */
}

Borders

.box {
    border: 2px solid #ccc; /* Light gray border */
    border-radius: 5px; /* Rounded corners */
    padding: 15px; /* Space inside the box */
}

Media Queries (Responsive Design)

Media queries allow you to apply CSS rules based on the device's characteristics, such as screen width.

@media (max-width: 768px) {
    body {
        background-color: #fff; /* Changes background color on smaller screens */
    }

    .container {
        grid-template-columns: 1fr; /* Stacks items in a single column */
    }

    h1 {
        font-size: 24px; /* Adjusts heading size on smaller screens */
    }
}

Styling Forms

input[type="text"] {
    width: 100%; /* Full width of its container */
    padding: 10px; /* Space inside the input field */
    border: 1px solid #ddd; /* Light border */
    border-radius: 4px; /* Rounded corners */
}

button {
    background-color: #007bff; /* Blue background */
    color: white; /* White text */
    border: none; /* No border */
    padding: 10px 20px; /* Space inside the button */
    border-radius: 4px; /* Rounded corners */
    cursor: pointer; /* Pointer cursor on hover */
}

button:hover {
    background-color: #0056b3; /* Darker blue on hover */
}

Accessibility and Contrast

body {
    color: #333; /* Dark text color for readability */
    background-color: #fff; /* White background */
}

a {
    color: #007bff; /* Blue link color */
}

a:hover {
    color: #0056b3; /* Darker blue on hover */
    text-decoration: underline; /* Underline on hover */
}

Conclusion

These notes cover fundamental CSS concepts and techniques to help you style web pages effectively. By understanding and applying these properties, you can create visually appealing and well-structured web content.
