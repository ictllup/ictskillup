Computer System

A computer system is a series of interrelated computer components working together to perform a task.

Components of a Computer System

  1. Computer Hardware
    • Input devices (keyboard, mouse)
    • Processing devices (Microprocessor Chip)
    • Storage devices (Hard disks, CDs)
    • Output devices (monitor, printer)
  2. Software
    • System software (Operating system, Utility programs, Device drivers, Programming languages)
    • Application software (Computer games, calculator, Media Player)
  3. Human Ware/Users
    • Ordinary User (someone without technical knowledge of computers)
    • Professional User (a person with formal education in computer technology)
  4. Data
    • Unprocessed data
    • Processed data (Information)
  5. Procedures
  6. Communication

Computer Parts and Their Uses

Part Description Examples
Central Processing Unit (CPU) Performs calculations and executes instructions Intel Core i5, AMD Ryzen 7
Motherboard Main circuit board of the computer ASUS, MSI, Gigabyte
Memory (RAM) Temporary storage for data and applications 8GB, 16GB, 32GB
Storage Drive Holds operating system, programs, and data Hard Disk Drive (HDD), Solid State Drive (SSD)
Power Supply Converts AC power from the wall to DC power for the computer EVGA, Corsair, Seasonic
Graphics Card Controls the output display and handles graphics processing NVIDIA GeForce, AMD Radeon
Sound Card Handles audio output and processing Built-in audio, dedicated sound cards like Sound Blaster
Network Card Connects the computer to a network or internet Ethernet, Wi-Fi
Case Outer casing of the computer Fractal Design, NZXT, Corsair
Cooling System Helps keep the computer components at a safe temperature Fans, Liquid Cooling Systems
Monitor Displays the output of the computer LED, LCD, OLED
Keyboard Input device for typing and commands Wired, wireless, mechanical
Mouse Input device for navigation and selection Wired, wireless, optical, laser
Speakers Output device for audio Built-in, external speakers
