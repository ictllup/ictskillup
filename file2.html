File Extensions Notes

What is a file extension?

A file extension is a set of characters added to the end of a file name to indicate the file's format or type.

Common File Extensions

Document files:

  • .docx (Microsoft Word document)
  • .pdf (Portable Document Format)
  • .txt (Plain text file)

Image files:

  • .jpg or .jpeg (Joint Photographic Experts Group image)
  • .png (Portable Network Graphics image)
  • .gif (Graphics Interchange Format image)

Audio files:

  • .mp3 (MPEG Audio Layer 3 file)
  • .wav (Waveform Audio File Format)

Video files:

  • .mp4 (MPEG-4 video file)
  • .avi (Audio Video Interleave file)

Compressed files:

  • .zip (Zip archive file)
  • .rar (RAR archive file)

Executable files:

  • .exe (Executable file for Windows)
  • .app (Executable file for macOS)

Other notable file extensions:

  • .html or .htm (Hypertext Markup Language file)
  • .css (Cascading Style Sheets file)
  • .js (JavaScript file)
  • .csv (Comma Separated Values file)

Remember:

  • File extensions can be hidden or shown in file names, depending on the operating system settings.
  • Some files may have multiple extensions (e.g., file.tar.gz).
  • Be cautious when opening files with unknown extensions, as they may contain malware.
