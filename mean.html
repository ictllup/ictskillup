Meaning of ICT

TERMINOLOGIES DEFINED

There are many terminologies used in the field of computing. Below are some of the basic ones:

1. Information and communication Technology (ICT) It is the branch of engineering that deals with the use of computers and telecommunications to retrieve, store and transmit information.

2. The term information technology (IT) is used to describe the use of computers in business, education and everyday life.

3. Data entry It is the act of in putting data into the computer. This is done by the help of input devices like keyboard, mouse etc.

4. Byte A byte is a unit of computer memory or digital information that consists of eight binary digits (bits). NB. 1 byte stores 1 character.

5. Binary The number system used to represent digital information in computers where data is expressed by combinations of the digits 0 and 1, corresponding to power states "off" and "on" respectively.

6. Computer Memory This acts as a store for digital Data, processed information and programs in a computer system.

7. RAM (Random Access Memory) is temporary and volatile working memory. It holds the tasks being processed

What is a computer?

  • computer is an electronic device that can accept data input, process it according to some specified instructions, output the information, and store the results for future use.

    • The term computer is derived from a Latin word "Computare" which means to count or to calculate.

    parts of a computer

    A computer comprises of two major parts; System unit and Peripheral devices.

    1. A System Unit
    2. A Peripheral device

    A System Unit (A Computer case):

    This is the metallic box that houses all the internal computer components

    These include

    1. Motherboard
    2. CPU
    3. Data buses
    4. Adapter cards
    5. Expansion slots
    6. power supply unit
    7. heat sink and the hard drive

    A Peripheral device

    refers to any computer hardware component that connects externally to the motherboard or system unit and can communicate with the CPU

    Examples include

    1. keyboard
    2. Mouse
    3. Scanner
    4. Monitor
    5. Printer

    Other parts

    • a. A computer monitor

      • This is a display device that outputs images and text on the screen to be viewed by the user.

    • b. A Keyboard

      • This is an input device that keys text and commands into the computer

    • c.A Mouse

      • This is a handheld pointing input device that controls a cursor on the computer screen and inputs data by clicking

    • d. Power Cables (Cord)

      • This is the link between the power outlet on the wall and the power supply unit in the computer case.

    • e. Visual Graphics Adapter Cable (VGA)

      • This is a cable that connects to the visual graphics card on the motherboard and allows display devices to show graphics (images)

      factors that determine the type of a computer

      In general, a computer's type is determined by the following seven factors:

      • a. The type or nature of the CPU
      • b. Amount of main memory the CPU can use
      • c. The Capacity of storage devices
      • d. The Speed of output devices
      • e. Processing speed in millions of instructions per second (MIPS
      • f. Number of users that can access the computer at one time
      • g. Cost of the computer: Sincerely, many users look at the asking price of a computer to determine its type. It is known that the higher the price the better the computer set a user gets.
